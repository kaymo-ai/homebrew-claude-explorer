# Homebrew Tap for Claude Explorer

This is the official Homebrew tap for [Claude Explorer](https://github.com/kaymo-ai/claude-explorer).

## Installation

```bash
brew install kaymo-ai/claude-explorer/claude-explorer
```

## What is Claude Explorer?

An interactive dashboard for exploring your Claude Code session history, settings, plans, and usage statistics.

See the [main repository](https://github.com/kaymo-ai/claude-explorer) for more details.
